== Description ==
This plugin enables posts to be published from Kapost.com to your WordPress blog.

The following capabilities are available:

- Author bylines and bios
- SEO integrations with popular plugins such as Yoast
- Draft previews

For more information, visit the [Kapost](http://www.kapost.com) and [Kapost Help](http://help.kapost.com) websites.

== Installation ==

1. Upload all the files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
